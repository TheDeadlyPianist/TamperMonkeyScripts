Auth Login Helper - Credentials Helper

Install

Why use the helper?

Inputting user credentials can be a long, tedious and painful process. What would help alleviate that process? Being able to associate a collection of credentials to a simple press of a button.

This helper, with the most minimal of set-up (the same as it would take to just fill in the credentials), you can save yourself at least 10 seconds every single time you use the Auth login stub.

Hit that install button and start saving time (for free!) today!

Features

  • Easily store specific credentials for different services.
  • Will automatically fill in the NINO field
  • Will automatically change the Affinity Group and Confidence Level dropdowns for you.
  • Will automatically fill in the cred fields with what you have supplied.
  • If you have provided more credentials than what is displayed by default, it will automatically add extra rows for you.
  • Provides different input fields depending on if you are making an Individual or Agent.

How to Use

This script adds two buttons above the enrolment key fields:

  • Create User Group
  • Create New User

Create User Group

A user group is, as the name suggest, a collection of users. A user must be placed in a group.

User groups are extremely easy to make.

Step 1

When you click on the "Create User Group", you get the new user group form.
As with all forms, the submit button submits it, and the cancel button closes it down.



Step 2

Enter a name and hit submit. That's all you need to do to create a user group.

Create User

Creating a User is a little more work than a user group, but no more effort than filling in the Auth Login form anyway.

There is a slightly different view depending on if you're filling in an Individual or an Agent.


Step 1


Individual



Agent



The only fields here worth mentioning are the "User Group" and "User Name" fields. The group is a dropdown of the user groups you have created. The user name is what the collection of credentials will be listed as in your user group.
The rest of the fields relate directly to the fields on the Auth Login form.
Simply fill them in and hit submit.
